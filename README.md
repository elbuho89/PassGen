# PassGen Pro 🔒 | ELBUHO89

Una extensión de navegador moderna, ligera y ultra segura para generar contraseñas con entropía criptográfica real. Soporta múltiples alfabetos y está diseñada bajo estándares de código abierto.

## ✨ Características

- **Seguridad Criptográfica:** Utiliza la `Web Crypto API` (`window.crypto.getRandomValues`) para garantizar que las contraseñas no sean predecibles.
- **Soporte Multilingüe:** Genera claves utilizando caracteres latinos, símbolos, cirílico, chino y japonés.
- **Mezcla Robusta:** Implementa el algoritmo de mezcla Fisher-Yates para asegurar una distribución uniforme de caracteres.
- **Privacidad Total:** No requiere permisos de red. Todo se genera localmente en tu navegador.
- **Acceso Rápido:** Copiado automático al portapapeles tras la generación.

## 🚀 Instalación (Modo Desarrollador)

Como esta es una extensión de código abierto, puedes instalarla manualmente:

1.  **Descarga** este repositorio como un archivo ZIP y descomprímelo.
2.  Abre tu navegador (Chrome, Edge o Brave) y ve a la dirección: `chrome://extensions/`.
3.  Activa el **"Modo de desarrollador"** (Developer mode) en la esquina superior derecha.
4.  Haz clic en el botón **"Cargar descomprimida"** (Load unpacked).
5.  Selecciona la carpeta donde descomprimiste los archivos.
6.  ¡Listo! Fija el icono del candado en tu barra de herramientas.

## 🛠️ Tecnologías utilizadas

- **HTML5 / CSS3:** Interfaz moderna con variables CSS y diseño adaptativo.
- **JavaScript (Vanilla):** Lógica pura sin dependencias externas para mayor seguridad.
- **Manifest V3:** Cumple con los últimos estándares de seguridad y rendimiento de las extensiones de Chrome.
